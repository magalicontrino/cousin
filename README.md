# COUSIN

Annuaire mobile du **réseau d'aide aux personnes à Bruxelles**, pour les équipes de travail social.

Application web autonome (un seul fichier `index.html`, sans serveur) : elle fonctionne hors ligne et enregistre les données sur l'appareil.

## Fonctionnalités

- Ressources classées par **domaine** : Médical, Social, Éducatif, Juridique, Droit des étrangers, Santé mentale, Logement, Urgences, et des **Méthodes de travail** pour l'équipe.
- Navigation en **pavés** d'un domaine à l'autre.
- Section **Médical classée par commune** (les 19 communes bruxelloises).
- Les **19 CPAS** avec coordonnées vérifiées (appel direct, itinéraire).
- Recherche, **favoris**, ajout / modification / suppression de fiches.
- **Sauvegarde / import** (JSON) pour ne rien perdre.

## Utilisation

Ouvrir `index.html` dans un navigateur. Sur téléphone : ouvrir la page, puis « Ajouter à l'écran d'accueil ».

## Identité visuelle

Fond noir, accent or, icônes pleines — logo COUSIN.
